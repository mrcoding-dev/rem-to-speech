## ğŸ¯ How to Use

### Quick Start

1. **Select text** in RemNote (drag to select or select multiple Rems)
2. **Click the "Leer" tab** that appears above the selection
3. **Press "Leer Texto"** button
4. **Listen!** A floating control widget appears while reading

### Selection Menu (Recommended)

When you select text or Rems in RemNote, you'll see a **tab labeled "Leer" (or "Read")**:

- Click the tab to open the selection menu
- Press the **"Leer Texto"** button with the speaker icon
- The reading starts immediately
- A floating widget appears for control

### Using Commands

You can also use the command palette:

1. Select text or Rems
2. Open command palette (Ctrl/Cmd + K)
3. Type **"Leer Texto Seleccionado"** or **"Read Selected Text"**
4. Press Enter

### Keyboard Shortcuts (Optional)
